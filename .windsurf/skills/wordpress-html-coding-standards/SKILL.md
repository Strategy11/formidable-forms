## Overview

These HTML coding standards ensure well-formed, accessible markup across the WordPress ecosystem.

**When to apply:** Any HTML in WordPress Core, plugins, themes, or templates.

**Reference:** [WordPress HTML Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/html/)
